# 1. 数组下标表示指针偏移量

```c
int a[10] = { ... };
int b = a[3]; // 也可以写作 3[a]
```

# 2. 方括号和花括号可以用组合括号代替

早期键盘没有方括号和花括号，因此这种方式作为向后兼容保留了下来（部分编译器可能不支持）。

- `[` 可以替换为 `<:`
- `]` 可以替换为 `:>`
- `{` 可以替换为 `<%`
- `}` 可以替换为 `%>`

# 3. `sizeof` 是一个关键字而非函数

`sizeof(a)` 和 `sizeof a` 是一样的。
